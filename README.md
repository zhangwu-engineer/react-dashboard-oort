# react-ui
## setting up dev environment

Run setup
```shell script
$ npm install
```

```shell script
$ npm build-all
```

Setup environment variables
```shell script
$ export REACT_APP_OKTA_ISSUER='foo'
$ export REACT_APP_OKTA_CLIENT_ID='bar'
$ export REACT_APP_ACTIVITY_LOG_API_URL='foobar'
```
Start development server
```shell script
$ npm start-all
```
